# CenX 浏览器

一个功能丰富、持续迭代的 Android 浏览器，集成了 AI 能力与多种实用工具。

## 🛠️ 主要功能一览

| 功能模块 | 描述 |
|---------|------|
| 书签管理 | 收藏与管理常用网页 |
| 历史记录 | 浏览历史留存与查看 |
| 源码查看 | 查看网页源代码 |
| 无痕模式 | 隐私浏览，不留痕迹 |
| 资源嗅探 | 探测并下载页面资源 |
| 全屏模式 | 沉浸式浏览体验 |
| 阅读模式 | 清爽阅读，去除干扰 |
| 小说阅读器 | 专属小说阅读体验 |
| 扫一扫 | 扫码快速访问 |
| 下载管理器 | 统一管理下载文件 |
| AI 聊天 | 基于 OpenAI 格式的自定义 AI |
| AI 翻译 / 总结 / 解释 | 智能辅助阅读 |
| 秘塔 AI 搜索 | 内置 AI 搜索引擎 |
| 工具箱 | URL转换、Base64、色卡、UA、IP查询 |
| 动态取色 | Android 12+ 动态主题色 |
| 实验室功能 | 站点鉴权、Cookie Manager |
| 自定义快捷入口 | 主页添加常用链接，带动画效果 |
